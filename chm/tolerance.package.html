tolerance-package(tolerance)R Documentation

Functions for Calculating Tolerance Intervals

Description

A collection of functions for calculating [100(1-alpha)%, 100(P)%] tolerance intervals, which are intervals with 100(1-alpha)% confidence of covering 100(P)% of the population of interest.
